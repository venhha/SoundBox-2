# SoundBox-2
Phần mềm nghe nhạc trên điện thoại android có các tính năng:
- Load nhạc từ FireBase (Google)
- Tải nhạc lên FireBase --người khác có thể nghe được (yêu cầu đăng ký tài khoản)
- Phát nhạc, chuyển tiếp, lặp lại,...
